# SmartWhoopGate32

## MQTT Reference

- LED
  - ON
    - `LED ON`
  - OFF
    - `LED OFF`
  - PARTY
    - `LED PARTY`
  - MODE
    - `LED MODE <11>`
- MAXRSSI
  - AUTORESET
    - ON
      - `MAXRSSI AUTORESET ON`
    - OFF
      - `MAXRSSI AUTORESET OFF`
  - RESET
    - `MAXRSSI RESET`
  - SET
    - `MAXRSSI SET [1] <2500>`
- EEPROM
  - SET
    - NAME
      - `EEPROM SET NAME <Gate10>`
    - NETWORK
      - []
        - `EEPROM SET NETWORK [0] <ssid:pass:mqtt>`
- UPDATE
  - `UPDATE`
- RESTART
  - `RESTART`
